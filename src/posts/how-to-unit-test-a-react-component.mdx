What we are going to cover in this post:

- How to refactor your React components to make them easier to test
- How to make assertions on the output of your component based on the props provided
- How to interact with a React component in your tests to assert expected behaviour
- How to mock 3rd party and local modules within your tests

What we are **not** going to cover in this post:

- How to run snapshot tests against your React components
- How to create integration tests for your React components
- How to create end-to-end tests for your React components
